SLT类问题

指令 指令描述
slti if ((rs)<(Sign-Extend)immediate) then (rt)←1; else (rt)←0
sltiu if ((rs)<(Zero-Extend)immediate) then (rt)←1; else (rt)←0
slt if (rs< rt) rd=1 else rd=0;

lui (rt)←immediate <<16 & 0FFFF0000H

处理了所有根据比较结果赋值的指令和luiluilui指令，虽然此处对subsubsub指令也做了处理，但是由于subsubsub指令并不是比较指令，StfmdStfmdStfmd为0，所以并不会将处理结果给ALU_ResultALU\_ResultALU_Result赋值，因此不会影响最终输出。
